# Simple Webpage App

A minimal static webpage consisting of `index.html`, `app.js`, and `style.css`.

## Overview

- **What it is:** A small front-end app (static HTML/CSS/JS) demonstrating a simple webpage project.
- **Files:** [index.html](index.html), [app.js](app.js), [style.css](style.css)

## Features

- Static single-page layout
- Client-side JavaScript in `app.js`

## Run / Preview

1. Open `index.html` directly in your browser, or serve the folder with a simple static server.

Quick local server options:

```
# Python 3
python3 -m http.server 8000

# Node (if you have http-server)
npx http-server -c-1
```

Then open http://localhost:8000 in your browser.

## Development

- Use the VS Code Live Server extension for fast reloads, or the commands above.
- Edit `app.js` and `style.css` to modify behavior and styles.
